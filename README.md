# Анализ набора данных Synthetic Clinical Tabular Dataset

## Описание проекта
Курсовая работа по первичному анализу табличных медицинских данных. 
Цель: провести разведочный анализ данных (EDA), выявить закономерности и визуализировать результаты.

## Структура репозитория
- `data/` — исходный датасет (CSV).
- `images/` — сгенерированные графики (11 визуализаций согласно заданию).
- `analysis.py` — основной скрипт на Python для генерации графиков.
- `requirements.txt` — зависимости проекта.
